# Оптимизация изображений для сайта Halcyon

## Оптимальные параметры

Для обеспечения наилучшей производительности сайта рекомендуется использовать следующие параметры при работе с изображениями:

1. **Форматы изображений:**
   - Используйте WebP для сложных изображений (фотографии, иллюстрации)
   - Используйте SVG для векторной графики (логотипы, иконки)
   - Используйте AVIF для наилучшего качества при меньшем размере (если поддержка браузеров позволяет)

2. **Размеры изображений:**
   - Используйте несколько размеров для одного изображения для разных устройств
   - Не загружайте изображения с разрешением выше, чем требуется для отображения

3. **Ленивая загрузка (Lazy Loading):**
   - Используйте атрибут loading="lazy" для изображений ниже по потоку

## Пример использования изображений в Astro

```astro
---
import { Image, Picture } from 'astro:assets';
import exampleImage from '../assets/images/example.jpg';
---

<!-- Использование компонента Image -->
<Image 
  src={exampleImage} 
  alt="Описание изображения" 
  width={800} 
  height={600} 
  loading="lazy" 
  formats={['avif', 'webp', 'jpeg']}
/>

<!-- Использование Picture для нескольких форматов -->
<Picture 
  src={exampleImage} 
  alt="Описание изображения" 
  widths={[400, 800, 1200]} 
  formats={['webp', 'jpeg']}
/>
```

## Рекомендуемые инструменты для оптимизации

- ImageOptim (Mac)
- FileOptimizer (Windows)
- Squoosh (онлайн-редактор от Google)
- Tinypng

## Проверка производительности

Для проверки влияния изображений на производительность сайта используйте:

- Google PageSpeed Insights
- Lighthouse
- WebPageTest
- GTmetrix

После добавления изображений на сайт, следуйте этим рекомендациям для обеспечения высокой производительности.
