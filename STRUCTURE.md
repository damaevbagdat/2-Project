# Структура проекта Halcyon

## Обзор

Этот файл описывает полную структуру каталогов и файлов проекта сайта юридической фирмы "Halcyon". Он предназначен для быстрого ознакомления с организацией файлов проекта.

## Структура

```
my-first-project/
├── .github/
│   └── copilot-instructions.md          # Инструкции для AI-ассистентов
├── .gitignore                           # Файлы и папки, исключенные из Git
├── README.md                           # Основное описание проекта
├── Requirements.md                     # Техническое задание
├── ROADMAP.md                          # Дорожная карта разработки
├── Qwen.md                             # Лог работы Qwen AI-агента
├── STRUCTURE.md                        # Этот файл
├── package.json                        # Зависимости и скрипты npm
├── astro.config.mjs                    # Конфигурация Astro
├── tsconfig.json                       # Конфигурация TypeScript
├── .prettierrc                         # Конфигурация Prettier
├── [lang]/                             # Redirects для корневых страниц
│   ├── index.astro                     # Redirect на /ru/
│   ├── about.astro                     # Redirect на /ru/about
│   └── contact.astro                   # Redirect на /ru/contact
├── src/
│   ├── components/                     # Переиспользуемые UI-компоненты
│   │   ├── FadeIn.astro                # Компонент для анимаций появления при скролле
│   │   ├── Header.astro                # Компонент шапки сайта
│   │   ├── Footer.astro                # Компонент подвала сайта
│   │   ├── Hero.astro                  # Компонент главного экрана
│   │   └── Advantages.astro            # Компонент преимуществ
│   ├── layouts/                        # Макеты страниц
│   │   └── BaseLayout.astro            # Основной макет сайта
│   ├── assets/                         # Изображения, иконки, шрифты
│   ├── content/                        # Контент в формате Markdown
│   │   ├── about.md                    # Legacy контент страницы "О компании"
│   │   ├── contact.md                  # Legacy контент страницы "Контакты"
│   │   ├── config.ts                   # Конфигурация коллекций контента
│   │   ├── pages/                      # Коллекция страниц для многоязычности
│   │   │   ├── about-en.md             # Контент страницы "О компании" (English)
│   │   │   ├── about-kz.md             # Контент страницы "О компании" (Kazakh)
│   │   │   ├── about-ru.md             # Контент страницы "О компании" (Russian)
│   │   │   ├── contact-en.md           # Контент страницы "Контакты" (English)
│   │   │   ├── contact-kz.md           # Контент страницы "Контакты" (Kazakh)
│   │   │   └── contact-ru.md           # Контент страницы "Контакты" (Russian)
│   │   ├── services/                   # Коллекция "Услуги"
│   │   │   ├── corporate-law.md        # Описание услуги "Корпоративное право"
│   │   │   └── litigation.md           # Описание услуги "Судебная практика"
│   │   └── blog/                       # Коллекция "Блог"
│   │       └── new-legal-requirements-2025.md  # Пример статьи блога
│   ├── styles/                         # Глобальные стили
│   │   └── global.css                  # Глобальные стили и CSS-переменные
│   └── pages/                          # Страницы сайта
│       ├── index.astro                 # Redirect на /ru/
│       ├── about.astro                 # Redirect на /ru/about
│       ├── contact.astro               # Redirect на /ru/contact
│       ├── [lang]/                     # Динамические маршруты для многоязычности
│       │   ├── index.astro             # Динамический маршрут для главной страницы
│       │   ├── about.astro             # Динамический маршрут для страницы "О компании"
│       │   ├── contact.astro           # Динамический маршрут для страницы "Контакты"
│       │   └── services/               # Динамические маршруты для страниц услуг
│       │       ├── index.astro         # Главная страница услуг в разных языках
│       │       └── [slug].astro        # Динамический маршрут для страниц услуг
│       └── services/                   # Legacy страницы услуг (для редиректа)
│           ├── index.astro             # Redirect на /ru/services
│           └── [slug].astro            # Redirect на /ru/services/slug
└── node_modules/                       # Зависимости проекта (исключены из Git)
```

## Описание ключевых файлов и папок

### .github/

- `copilot-instructions.md` — файл с инструкциями для AI-ассистентов, работающих с проектом

### Корневая директория

- `README.md` — основное описание проекта
- `Requirements.md` — техническое задание на разработку сайта
- `ROADMAP.md` — пошаговый план разработки сайта с использованием Astro
- `Qwen.md` — лог работы Qwen AI-агента
- `STRUCTURE.md` — этот файл со структурой проекта
- `package.json` — файл конфигурации npm с зависимостями и скриптами
- `astro.config.mjs` — конфигурационный файл Astro
- `tsconfig.json` — конфигурационный файл TypeScript
- `.prettierrc` — настройки форматирования кода с помощью Prettier
- `.gitignore` — список файлов и папок, исключенных из Git
- `[lang]/index.astro` — редирект главной страницы на язык по умолчанию
- `[lang]/about.astro` — редирект страницы "О компании" на язык по умолчанию
- `[lang]/contact.astro` — редирект страницы "Контакты" на язык по умолчанию

### src/components/

- `FadeIn.astro` — компонент для анимаций плавного появления элементов при скролле
- `Header.astro` — компонент шапки сайта с логотипом, навигацией и переключателем языков
- `Footer.astro` — компонент подвала сайта с контактной информацией и социальными ссылками
- `Hero.astro` — компонент главного экрана (hero section)
- `Advantages.astro` — компонент с отображением ключевых преимуществ компании

### src/layouts/

- `BaseLayout.astro` — основной макет сайта, включающий шапку и подвал

### src/styles/

- `global.css` — файл с глобальными стили и CSS-переменными для цветов и шрифтов согласно ТЗ

### src/pages/

- `index.astro` — редирект главной страницы на язык по умолчанию
- `about.astro` — редирект страницы "О компании" на язык по умолчанию
- `contact.astro` — редирект страницы "Контакты" на язык по умолчанию
- `[lang]/` — директория динамических маршрутов для многоязычности
  - `index.astro` — динамический маршрут для главной страницы с поддержкой языков
  - `about.astro` — динамический маршрут для страницы "О компании" с поддержкой языков
  - `contact.astro` — динамический маршрут для страницы "Контакты" с поддержкой языков
  - `services/` — директория для страниц услуг с поддержкой языков
    - `index.astro` — главная страница услуг с поддержкой языков
    - `[slug].astro` — динамический маршрут для отдельных страниц услуг с поддержкой языков
- `services/` — устаревшие страницы услуг (для редиректа)
  - `index.astro` — редирект на /ru/services
  - `[slug].astro` — редирект на /ru/services/slug

### src/assets/

- Папка для хранения изображений, иконок и шрифтов (пока пуста)

### src/content/

- `about.md` — устаревший контент страницы "О компании" в формате Markdown
- `contact.md` — устаревший контент страницы "Контакты" в формате Markdown
- `config.ts` — конфигурационный файл для коллекций контента (услуги и блог)
- `pages/` — коллекция многоязычных страниц
  - `about-en.md` — контент страницы "О компании" на английском языке
  - `about-kz.md` — контент страницы "О компании" на казахском языке
  - `about-ru.md` — контент страницы "О компании" на русском языке
  - `contact-en.md` — контент страницы "Контакты" на английском языке
  - `contact-kz.md` — контент страницы "Контакты" на казахском языке
  - `contact-ru.md` — контент страницы "Контакты" на русском языке
- `services/` — коллекция контента для услуг юридической фирмы
  - `corporate-law.md` — описание услуги "Корпоративное право"
  - `litigation.md` — описание услуги "Судебная практика"
- `blog/` — коллекция контента для публикаций и новостей
  - `new-legal-requirements-2025.md` — пример статьи блога

- Реализация интерактивности и анимаций
