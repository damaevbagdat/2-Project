# План настройки сервера для проекта "Halcyon"

Этот документ описывает шаги по настройке выделенного сервера (VDS/VPS) для развертывания сайта, созданного на Astro.

**Данные сервера:**
- **IP-адрес:** `18.196.166.160`
- **Пользователь:** `ubuntu`

---

### Этап 1: Первоначальная настройка и безопасность

- [x] **1.1. Подключение к серверу:** Подключиться к серверу по SSH, используя предоставленные учетные данные.
- [x] **1.2. Смена пароля:** **(Критически важно!)** Немедленно сменить пароль пользователя `ubuntu`.
- [x] **1.3. Обновление системы:** Обновить список пакетов и сами пакеты до последних версий.
- [x] **1.4. (Рекомендуется) Настройка Firewall:** Настроить базовый брандмауэр (например, UFW), открыв порты для SSH (22), HTTP (80) и HTTPS (443).

### Этап 2: Установка необходимого ПО

- [x] **2.1. Установка Node.js:** Установить Node.js (версии 18.x) и npm. Рекомендуется использовать `nvm` (Node Version Manager) для удобства управления версиями.
- [x] **2.2. Установка Git:** Установить систему контроля версий Git.
- [x] **2.3. Установка веб-сервера Nginx:** Установить Nginx, который будет выступать в роли веб-сервера и прокси для нашего сайта.

### Этап 3: Развертывание проекта

- [x] **3.1. Клонирование репозитория:** Склонировать репозиторий проекта с GitHub на сервер.
- [x] **3.2. Установка зависимостей:** В папке проекта выполнить команду `npm ci` для установки всех необходимых пакетов.
- [x] **3.3. Сборка проекта:** Выполнить команду `npm run build` для сборки статической версии сайта. В результате появится папка `dist`.

- [x] **4.1. Создание конфигурации:** Создать новый конфигурационный файл для нашего сайта в директории Nginx.
- [x] **4.2. Настройка Nginx:** В конфигурации указать, что Nginx должен "слушать" 80 порт и отдавать файлы из папки `dist` нашего проекта.
- [x] **4.3. Перезапуск Nginx:** Применить новую конфигурацию, перезапустив сервис Nginx.

### Этап 5: Настройка домена и HTTPS

- [x] **5.1. Настройка DNS:** В настройках вашего домена `alash-zan.kz` создать A-запись, указывающую на IP-адрес сервера `18.196.166.160`.
- [x] **5.2. Настройка HTTPS:** Установить и настроить бесплатный SSL-сертификат от Let's Encrypt с помощью утилиты `certbot`, чтобы сайт работал по безопасному протоколу HTTPS.
