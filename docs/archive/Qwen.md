- **Frontend**: Mobile First approach for responsive design
- **Multi-language**: Support for three languages (KZ, RU, EN) with URL structure like `domain.com/ru/page`

## Technical Requirements (from Requirements.md and updated copilot instructions)
### Core Features
- Управление контентом через Markdown-файлы.
- Mobile-first responsive design
- Multi-language support (KZ, RU, EN)
- Contact forms with integration to email/Telegram
- Integration with messengers (WhatsApp, Telegram)
- Interactive elements and smooth animations

### Design Requirements
- Style: Strict, minimalist, inspiring trust
- Color palette:
  - Primary: Deep blue (#0A2342) or dark gray (#2D3748)
  - Accent: Gold (#D4AF37) or turquoise (#4FD1C5)
  - Background: White (#FFFFFF) or light gray (#F7FAFC)
- Animations: Smooth and unobtrusive, without affecting performance
- Typography: Clean with clear hierarchy for good readability

### Technical Requirements
- SEO optimization (semantic markup, meta tags, clean URLs, fast loading)
- Google Analytics and Yandex.Metrica integration
- Compliance with Kazakhstan's personal data protection law
- Personal data processing consent fields in all forms
- Semantic HTML structure for accessibility and SEO
