ИНСТРУКЦИЯ: Как добавить gose.kz в hosts файл
================================================

ПРОБЛЕМА:
Когда ты открываешь http://194.32.142.237 в браузере, Nginx не знает какой сайт показать
(gose.kz или alash-zan.kz), поэтому показывает дефолтную страницу.

РЕШЕНИЕ:
Временно добавь gose.kz в файл hosts на своем компьютере.

ШАГИ:

1. Открой Блокнот от имени администратора:
   - Нажми Windows + S
   - Введи "Блокнот"
   - Правый клик → "Запуск от имени администратора"

2. В Блокноте:
   - Файл → Открыть
   - Вставь путь: C:\Windows\System32\drivers\etc\hosts
   - Измени фильтр на "Все файлы (*.*)"
   - Открой файл hosts

3. Добавь в конец файла:

194.32.142.237 gose.kz www.gose.kz

4. Сохрани файл (Ctrl+S)

5. Закрой Блокнот

6. Открой браузер и перейди на: http://gose.kz

ГОТОВО! Теперь сайт должен открываться.

=================================================

АЛЬТЕРНАТИВНЫЙ СПОСОБ (если hosts не работает):

Используй расширение для браузера, которое подменяет Host заголовок,
или дождись настройки настоящего DNS для домена gose.kz.

=================================================

ДЛЯ УДАЛЕНИЯ (после настройки DNS):

Просто удали строку "194.32.142.237 gose.kz www.gose.kz" из hosts файла.
